WikiHTML

WikiHTML is a Java library and executable standalone application that converts a file in wiki format to an HTML file.

Source code

To checkout and compile the project, use:

$ git clone https://github.com/julianmendez/wikihtml.git
$ cd wikihtml
$ mvn clean install
